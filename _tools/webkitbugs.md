## What it does

The official bug tracker for the WebKit project, which powers Safari on Apple platforms. Tracks rendering bugs, JavaScript engine issues (JavaScriptCore), and web platform feature requests. Filing a bug here is the primary way to report a Safari-specific compat issue or request implementation of a web standard.

## Who is it for

Web developers diagnosing Safari-specific layout or API bugs, WebKit contributors working on fixes, and web compat researchers tracking cross-browser interoperability gaps in the Safari/WebKit engine.

## Where to find it

[bugs.webkit.org](https://bugs.webkit.org)

## Who is maintaining it

Apple, with contributions from the WebKit open-source community including Igalia and other independent contributors.

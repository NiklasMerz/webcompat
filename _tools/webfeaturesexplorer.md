## What it does

Provides a browsable interface for the web-features dataset — listing every named web feature with its Baseline status, browser support, and links to MDN documentation and the underlying BCD data.

## Who is it for

Developers who want to explore what features are Baseline-supported, track newly available features, or get an overview of the web platform feature landscape without writing code against the web-features package directly.

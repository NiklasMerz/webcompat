## What it does

A config format and query engine for specifying which browser versions a project needs to support. A `.browserslistrc` file or `package.json` `browserslist` field is read by Babel, PostCSS, Autoprefixer, ESLint, Stylelint, and other build tools to automatically target the right output for the declared browser set.

## Who is it for

Frontend developers using build toolchains who want a single, consistent definition of browser targets shared across all their tools — so Babel, Autoprefixer, and eslint-plugin-compat all use the same browser list.

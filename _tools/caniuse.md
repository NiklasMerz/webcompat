## What it does

Provides browser support tables for web platform features — CSS, HTML, JavaScript APIs, and more — with a focus on quick, scannable answers. Shows which browser versions support a feature and at what point support was added.

## Who is it for

Web developers making decisions about which CSS properties or APIs to use, particularly when targeting a specific set of browser versions. Also heavily used by developers generating browser target configs via tools like browserslist.

## Where to find it

[caniuse.com](https://caniuse.com)

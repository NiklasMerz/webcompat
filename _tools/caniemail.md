## What it does

Shows HTML and CSS feature support across email clients — Gmail, Outlook, Apple Mail, Thunderbird, and others. Email rendering is significantly more fragmented than browser rendering, and this site maps that landscape.

## Who is it for

Email developers and marketers building HTML emails who need to know which CSS properties or HTML elements will actually render correctly across the email clients their audience uses.

## Where to find it

[caniemail.com](https://www.caniemail.com)

Data is also available on GitHub at [github.com/nicobytes/caniemail](https://github.com/HTeuMeuLeu/caniemail).

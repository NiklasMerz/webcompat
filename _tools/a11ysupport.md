## What it does

Provides compatibility tables for accessibility features — showing which browser and assistive technology combinations correctly support a given HTML element, ARIA role, or pattern. Functions as a "Can I Use" for AT support.

## Who is it for

Developers building accessible interfaces who need to know whether a specific ARIA pattern or semantic element is reliably announced by screen readers across browser and AT combinations.

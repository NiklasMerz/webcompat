## What it does

Defines a curated, named set of web platform features with stable identifiers, grouping related specs and APIs into units that make sense to developers. Provides the data layer that powers Baseline.

## Who is it for

Browser vendors and spec authors who need a shared vocabulary for web features, tool authors who want to reference features by stable ID, and documentation writers building feature-level compat summaries.

## Where to find it

[github.com/web-platform-dx/web-features](https://github.com/web-platform-dx/web-features)

The npm package `web-features` exposes the data for programmatic use.

## Who is maintaining it

The [web-platform-dx](https://github.com/web-platform-dx) working group, a cross-vendor effort involving Google, Mozilla, Apple, Microsoft, and others.

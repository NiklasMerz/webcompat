## What it does

The primary reference documentation for the web platform. Each API, CSS property, HTML element, and HTTP header has a page with a description, syntax, examples, and a detailed compat table sourced from BCD. MDN is also a major contributor back to the BCD data it consumes.

## Who is it for

Web developers of all experience levels — from looking up a specific method signature to understanding the full behaviour of a complex API. Also used by browser vendors and spec authors as a reference.

## Where to find it

[developer.mozilla.org](https://developer.mozilla.org)

## Who is maintaining it

Mozilla funds the infrastructure and a core editorial team. Open Web Docs employs technical writers who contribute content. Thousands of community contributors maintain individual pages.

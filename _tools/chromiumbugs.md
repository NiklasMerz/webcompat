## What it does

The official issue tracker for the Chromium project, covering bugs and feature requests for Chrome, ChromeOS, and the underlying Blink rendering engine. Tracks everything from rendering regressions and JavaScript engine bugs to web platform feature requests and standards implementation work. Issues can be starred to signal interest, and star counts are used as a proxy for developer demand when prioritising work.

## Who is it for

Web developers filing browser bugs or checking the status of a known issue, Chromium contributors triaging and fixing bugs, and web compat researchers tracking browser-specific breakage.

## Where to find it

[issues.chromium.org](https://issues.chromium.org) — the short alias [crbug.com](https://crbug.com) redirects there. Individual issues are accessible at crbug.com/\<number\>.

## Who is maintaining it

Google, with contributions from the broader Chromium open-source community.

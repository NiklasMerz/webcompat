## What it does

Browser built-in developer tooling — inspectors, consoles, network panels, performance profilers, and increasingly, compat warnings. Chrome DevTools, Firefox DevTools, and Safari Web Inspector all surface compatibility information directly in the authoring workflow, such as flagging unsupported CSS properties or deprecated APIs.

## Who is it for

Web developers building and debugging sites, who benefit from inline compat warnings rather than having to consult external reference sites.

## Where to find it

Built into every major browser. Opened with F12 or right-click → Inspect.

## Who is maintaining it

Each browser vendor maintains their own tools: Google (Chrome DevTools), Mozilla (Firefox DevTools), Apple (Web Inspector).

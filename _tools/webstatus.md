## What it does

A dashboard showing the implementation status of web platform features across browsers. For each feature it shows the Baseline status, which browsers support it, and links to WPT test results and the underlying web-features definition. It acts as a single place to answer "is this feature widely supported yet?"

## Who is it for

Web developers tracking when a feature reaches Baseline, browser engineers monitoring cross-browser implementation progress, and spec authors checking the adoption landscape for proposals they are working on.

## Where to find it

[webstatus.dev](https://webstatus.dev)

## Who is maintaining it

Maintained by Google as part of the web-platform-dx organisation, the same group that stewards the web-features repository and the Baseline definition.

## What it does

A JSON database of W3C, WHATWG, IETF, and other web-related specifications, providing stable short-hand references (e.g. `[HTML]`, `[CSS-COLOR-4]`) that tools can resolve to a full title, URL, and metadata. It is the canonical source of spec aliases used across the web standards ecosystem.

## Who is it for

Specification authors who need to cross-reference other specs, and tools like browser-specs and Reffy that need to resolve or complete the list of known web specifications.

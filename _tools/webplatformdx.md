## What it does

web-features is the package that describes Web platform features and provides Baseline status reports.

## Who is it for

Tool authors integrating feature-level compat data, contributors proposing new feature definitions, and anyone wanting to understand how individual BCD entries map to higher-level feature concepts.

## Where to find it

[github.com/web-platform-dx/web-features](https://github.com/web-platform-dx/web-features)

## Who is maintaining it

The web-platform-dx working group. Contributions are welcome via pull requests — particularly for adding feature definitions and mapping BCD keys to features.
